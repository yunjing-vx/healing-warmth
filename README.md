## ✨ 项目简介

愈见温暖是一个专为生病期间设计的智能陪伴助手，通过温暖的话语和科学的知识，帮助用户缓解生病时的焦虑情绪，提供贴心的情感支持和健康科普。让每一个生病的夜晚都有温暖相伴。

- 温馨提醒及时就医的重要性

### 📚 身体系统科普

## 📦 本地开发

### 环境要求
- Node.js 18+ 或 Bun
- 现代浏览器

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/yunjing-vx/healing-warmth.git
   cd healing-warmth
   ```

2. **安装依赖**
   ```bash
   bun install
   ```

3. **启动开发服务器**
   ```bash
   bun dev
   ```

4. **打开浏览器访问**
   ```
   http://localhost:5173
   ```

### 构建部署

```bash
# 构建生产版本
bun build

# 预览构建结果
bun preview
```
